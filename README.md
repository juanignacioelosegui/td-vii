# 📊 TD7 - Ingeniería de Datos

## 📚 Programa – Primer Semestre 2026

Materia orientada a comprender los fundamentos del almacenamiento, administración y procesamiento de datos, incluyendo tanto tecnologías tradicionales como desafíos modernos de Big Data.

## 🎯 Objetivos de la Materia

- Comprender los fundamentos e historia de las bases de datos
- Diseñar y optimizar bases de datos desde una perspectiva teórica y práctica
- Manejar conceptos de calidad, seguridad y gobernanza de datos
- Dominar el lenguaje SQL
- Introducirse en el procesamiento de grandes volúmenes de datos (Big Data)
- Conocer arquitecturas modernas de datos
- Entender tecnologías de almacenamiento y procesamiento distribuido
- Analizar bases de datos NoSQL y su contexto

---
